Compara.{release}.ncrna.nh.emf.gz 
  contains the tree for every ncRNA tree in Newick (New Hampshire) format

Compara.{release}.ncrna.nhx.emf.gz 
  contains the tree for every ncRNA tree in New Hampshire Extended (NHX) format with:
  [D] duplication tag -- Y or N
  [G] gene stable_id
  [T] NCBI taxon_id

Compara.{release}.ncrna.aln.emf.gz 
  contains the peptide alignment for every ncRNA tree in EMF Alignment format

Compara.{release}.ncrna.nt.fasta.gz 
  contains the nucleotide alignment for every ncRNA tree in FASTA format
  The token "//" is used to separate the alignments
